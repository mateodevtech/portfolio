Aperçu

Cette plateforme e-commerce a été développée dans le cadre du programme WeCode à Epitech. L'objectif : construire une expérience d'achat complète, du catalogue produit au paiement, avec une API REST robuste en backend.

Le frontend React communique avec un backend Spring Boot exposant les ressources produits, panier et commandes, le tout connecté à une base MySQL.
